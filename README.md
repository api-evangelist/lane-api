# Lane API (lane-api)

Lane offers a feature-rich, end-to-end order management system for asset leasing, loans and credit companies. The platform covers all aspects from conducting end-to-end sales to performing dealer and partner-related tasks and marketing-related activities. The system offers a variety of dashboards that provide vital information for dealers and partners while enabling quick order management and providing a way for users to record and submit a complete credit application for their clients.

## APIs

### Lane API

Lane is a pure-play SaaS product with a secure RESTful API that can be consumed by invoking the methods documented in the OpenAPI specification. The API spans order management, customer and dealer operations, dashboards, document generation, signatures, payments, fraud and credit checks, vehicle and inventory metadata, integrations, webhooks, and analytics.
